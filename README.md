①課題-じゃんけんアプリrich-
マツケンクソゲー

②課題内容（どんな作品か）
-上様を20秒以内に右から左に移動するゲームです。動く上様を押すと、前進・後退・待ち、になります。

③DEMO
https://ziko-yoshino.github.io/matukenrich2/

④工夫した点・こだわった点
-端についたら、音楽が切り替わり、クラッカーで上様ステッカーがふってくる
-回っている上様にホバーしたら大きくなる
-20秒以内に到達しなかったら強制終了する

④難しかった点・次回トライしたいこと(又は機能)
-馬が到達したら、反応する動作が難しかったです。
